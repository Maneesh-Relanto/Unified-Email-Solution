# Configuration Files

This folder contains build tool and framework configuration files.

## Files

- **vite.config.ts** - Vite bundler configuration for client build
- **vite.config.server.ts** - Vite configuration for server build
- **tailwind.config.ts** - TailwindCSS utility framework configuration
- **postcss.config.js** - PostCSS configuration (used by Tailwind)
- **components.json** - shadcn/ui components CLI configuration

## Note

The root folder contains stub files that import from here to maintain compatibility with build tools that expect configs in the root directory.
